Overview

Allows user to specify the input parameters to the Algorithmic Trading backend to discover the best strategies to make profits over a range of dates.

Choose the stock

Choose the stock you want to back test. Then, select the start and end date.

Choosing strategy

Type of strategies available:

  1. Single Indicator (Only using one indicator to back test)

Choose indicator(s)

List of indicators:

  • SMA (Baseline)
    • Buy if candle closes above the SMA
    • Sell if candle closes below the SMA.
  • SMA (Crossing)
    • Buy if faster SMA crosses above the slower SMA.
    • Sell if faster SMA crosses below the slower SMA.
  • RSI
    • Buy if RSI crosses upwards from oversold region.
    • Sell if RSI crosses downwards from the overbought region.

Number of Shares

The number of shares that will be traded. Currently fixed at 1000.
